# Code of Conduct

## Expected behavior
- Be respectful and direct.
- Focus on technical substance.
- Assume good intent, but keep review standards high.
- Report bugs and risks clearly.

## Unacceptable behavior
- Harassment or personal attacks.
- Intentionally disruptive behavior.
- Sharing secrets, credentials, or private data from the project.

## Enforcement
Project maintainers may remove comments, issues, or contributions that do not
follow these rules.
